# BeeOnline

BeeOnline is a **community-built, arcade-inspired multiplayer game** designed from the ground up with **online networking** and **cross-platform play** in mind.

The project is **inspired by the gameplay feel** of the arcade game *Killer Queen*, but is **not a remake, clone, or unofficial port**. BeeOnline is an independent project with original code, assets, and design decisions, focused on bringing similar high-energy, team-based gameplay into a modern online environment.

---

## Important Notice

**BeeOnline is a fan/community-driven project and is not affiliated with, sponsored by, or endorsed by the creators or rights holders of *Killer Queen*.**

- No original *Killer Queen* code, ROMs, assets, audio, fonts, branding, cabinet artwork, or proprietary materials are used.

---

## Project Goals

- Create a **fast, competitive, team-based arcade experience** that works well online.
- Add **networking and online multiplayer** as first-class features.
- Support community-hosted matches, events, and experimentation.
- Remain accessible to contributors and easy to build locally.

---

## Non-Goals

- This project is **not** a 1:1 recreation or emulation of any existing arcade game.
- This project does **not** attempt to replicate cabinet software, ROM behavior, or internal networking.
- This project will **not** use official game names, character names, assets, or branding.

---

## Features (Work in Progress)

- Online multiplayer
- Networked game state and synchronization
- Cross-platform support (Windows, macOS, Linux)
- HID controller input support
- Arcade-friendly gameplay loops

Planned / exploratory:
- Matchmaking & lobbies
- Spectator mode
- Dedicated server support
- Community tooling
